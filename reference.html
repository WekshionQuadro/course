Ключевые термины

Bot (Класс)
Главный класс для взаимодействия с Telegram Bot API. Все запросы к серверам Telegram происходят через экземпляры этого класса. Требует токен при инициализации.
Handler (Обработчик)
Асинхронная функция, которая реагирует на определенные события (апдейты) от Telegram. Хэндлеры срабатывают, когда пользователь пишет текст, нажимает кнопку или отправляет фото.
Dispatcher (Диспетчер)
Корневой элемент маршрутизации. Он принимает все входящие обновления и решает, какому именно хэндлеру их передать в зависимости от фильтров.
Router (Роутер)
Модульный обработчик. Позволяет разбивать логику большого бота на разные файлы.

Примеры реализации

Пример создания базового хэндлера

Для того чтобы бот ответил на текстовое сообщение пользователя, необходимо создать функцию-обработчик и обернуть ее в соответствующий декоратор.

from aiogram import Router, types
from aiogram.filters import Command

router = Router()

@router.message(Command("start"))
async def cmd_start(message: types.Message):
    # Бот ответит в тот же чат, откуда пришла команда /start
    await message.answer("Привет! Я бот.")

Обычные клавиатуры (ReplyKeyboardMarkup)

Это кнопки, которые заменяют стандартную клавиатуру на устройстве пользователя. Они отправляют текст от лица юзера.

from aiogram.types import ReplyKeyboardMarkup, KeyboardButton

@router.message(Command("menu"))
async def show_menu(message: types.Message):
    kb = ReplyKeyboardMarkup(
        keyboard=[[KeyboardButton(text="О нас"), KeyboardButton(text="Контакты")]],
        resize_keyboard=True # Делает кнопки компактными
    )
    await message.answer("Выберите пункт меню:", reply_markup=kb)

Инлайн-кнопки (Callbacks)

Работа с инлайн-кнопками (Callbacks):

Когда пользователь нажимает инлайн-кнопку под сообщением, Telegram отправляет апдейт типа CallbackQuery. Вы обязаны ответить на него с помощью callback.answer(), иначе кнопка у пользователя будет "висеть" в состоянии загрузки.

@router.callback_query(F.data == "buy_item")
async def handle_buy_button(callback: types.CallbackQuery):
    # 1. Убираем "часики" загрузки на нажатой кнопке
    await callback.answer("Товар добавлен в корзину!")
    
    # 2. Изменяем текст сообщения, к которому была прикреплена кнопка
    await callback.message.edit_text("Корзина обновлена. Что дальше?")

Справочник методов Telegram Bot API

Метод API Описание и параметры метода
Bot.send_message() Базовый метод отправки текстового сообщения. Требует явного указания чата.
Параметр Тип Описание
chat_id Integer/String ID чата или @username канала
text String Текст сообщения (от 1 до 4096 символов)
parse_mode String Тип разметки (HTML, MarkdownV2)
Message.answer() Алиас. Отправляет сообщение в тот же чат, не требуя ID.
Параметр Тип Описание
text String Текст ответа
reply_markup Keyboard Обычная или инлайн-клавиатура
Message.reply() Алиас. Отправляет сообщение с визуальным "ответом" (цитированием).
Параметр Тип Описание
text String Текст ответа
Bot.send_photo() Отправка изображения по ID файла, ссылке или с диска.
Параметр Тип Описание
chat_id Integer/String Целевой чат
photo InputFile/String Фотография для отправки
caption String Текст подписи (до 1024 символов)
Message.answer_photo() Алиас. Отправляет фото в текущий чат.
Параметр Тип Описание
photo InputFile/String Фотография
caption String Текст подписи к фото
Bot.send_video() Отправка видеофайлов (mp4).
Параметр Тип Описание
chat_id Integer/String Целевой чат
video InputFile/String Видеофайл
duration Integer Длительность видео в секундах
Bot.send_document() Отправка файлов любого типа (без сжатия).
Параметр Тип Описание
chat_id Integer/String Целевой чат
document InputFile/String Файл документа
Bot.delete_message() Удаление сообщения в чате (требуются права администратора).
Параметр Тип Описание
chat_id Integer/String ID чата, где находится сообщение
message_id Integer ID удаляемого сообщения
Bot.edit_message_text() Изменяет текст уже отправленного ботом сообщения.
Параметр Тип Описание
text String Новый текст сообщения
chat_id Integer/String Чат, где находится сообщение
message_id Integer ID редактируемого сообщения
CallbackQuery.answer() Ответ на нажатие инлайн-кнопки (снимает "часики").
Параметр Тип Описание
text String Текст всплывающего уведомления (toast)
show_alert Boolean Показывать ли уведомление как окно с кнопкой OK
CallbackQuery.message.answer() Отправляет новое сообщение в ответ на нажатие кнопки.
Параметр Тип Описание
text String Текст нового сообщения
Bot.send_chat_action() Показывает статус "печатает...", "загружает фото..." и т.д.
Параметр Тип Описание
chat_id Integer/String Целевой чат
action String Тип действия (typing, upload_photo, etc.)
Bot.ban_chat_member() Исключает (банит) пользователя из группы или канала.
Параметр Тип Описание
chat_id Integer/String ID группы или канала
user_id Integer ID блокируемого пользователя
until_date Integer Unix time окончания бана
Bot.restrict_chat_member() Ограничивает права пользователя (например, режим read-only).
Параметр Тип Описание
chat_id Integer/String ID группы
user_id Integer ID пользователя
permissions ChatPermissions Объект с набором разрешений
Bot.set_my_commands() Устанавливает меню команд (при вводе /).
Параметр Тип Описание
commands List Список объектов BotCommand
scope BotCommandScope Область видимости команд
Bot.send_media_group() Отправляет альбом из нескольких фото или видео.
Параметр Тип Описание
chat_id Integer/String Целевой чат
media List Список медиа-объектов (от 2 до 10 шт)
Message.forward() Алиас. Пересылает сообщение в другой чат с указанием автора.
Параметр Тип Описание
chat_id Integer/String ID чата, куда переслать сообщение
